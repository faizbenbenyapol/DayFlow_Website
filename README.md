# DayFlow - Productivity and Life Management Platform

ระบบบริหารจัดการประสิทธิภาพการทำงาน การวางแผนงาน บันทึกสุขภาพ และการบริหารจัดการการเงินแบบครบวงจร พัฒนาขึ้นมาเพื่อช่วยจัดระเบียบชีวิตประจำวันให้มีระบบและมีประสิทธิภาพสูงสุด

เวอร์ชัน: v1.1.1

> [!WARNING]
> สถานะการพัฒนา: ระบบนี้กำลังอยู่ในช่วงการพัฒนาอย่างต่อเนื่อง (Under Development) ฟังก์ชันการทำงานบางอย่างอาจมีการเปลี่ยนแปลงหรือปรับปรุงเพื่อความเสถียรยิ่งขึ้น

## ภาพรวมของระบบ (Project Overview)

DayFlow เป็นเว็บแอปพลิเคชันที่สร้างขึ้นด้วยสถาปัตยกรรมแบบ Custom PHP MVC Framework โดยไม่ได้ใช้เฟรมเวิร์กสำเร็จรูป เพื่อความยืดหยุ่นในการจัดการทรัพยากร ประสิทธิภาพในการประมวลผล และความปลอดภัยของข้อมูล ระบบได้รับการออกแบบส่วนติดต่อผู้ใช้ (UI/UX) ภายใต้แนวคิด Neumorphism และ Soft UI ซึ่งเน้นความเรียบหรู ทันสมัย สบายตา มีมิติผ่านการไล่ระดับเงา (Soft Shadows) และรองรับการแสดงผลทุกขนาดหน้าจออย่างสมบูรณ์ (Fully Responsive)

## คุณสมบัติหลักของระบบ (Core Features)

### 1. ระบบแดชบอร์ดหลัก (Dashboard)
- ศูนย์รวมข้อมูลสรุปประจำวัน โน้ตด่วน และสถานะของโครงการ
- หน้าจอแสดงผลแบบ Modular ที่สามารถจัดวางและเปิดปิด Widget ต่างๆ ได้อย่างอิสระตามความต้องการของผู้ใช้

### 2. ระบบจดบันทึกอัจฉริยะ (Notes)
- ระบบบันทึกข้อมูลในรูปแบบบล็อกอิสระ (Block-Based Editor) รองรับข้อความ ลิงก์ และรายการตรวจสอบ (Checklist)
- รองรับการปักหมุดบันทึกสำคัญ และการจัดหมวดหมู่ผ่านแท็ก (Tags)
- ระบบเข้ารหัสข้อมูลบันทึกส่วนตัว (Note Encryption) เพื่อความปลอดภัยของข้อมูลขั้นสูง

### 3. ระบบจัดการโครงการและงาน (Project & Task Management)
- บอร์ดบริหารงานในรูปแบบคัมบัง (Kanban Board) ที่รองรับการทำงานแบบลากและวาง (Drag-and-Drop)
- กำหนดระดับความสำคัญ (Priority) และวันส่งมอบงาน (Due Date) เพื่อการติดตามที่ชัดเจน

### 4. ระบบจับเวลาโฟกัส (Pomodoro Focus Timer)
- เครื่องมือจับเวลาการทำงานตามเทคนิค Pomodoro ช่วยเสริมสร้างสมาธิ
- ระบบบันทึกประวัติการใช้งาน (Session Logging) พร้อมแสดงข้อมูลสถิติเพื่อนำมาวิเคราะห์และปรับปรุงประสิทธิภาพการทำงาน

### 5. ระบบบันทึกสุขภาพและโภชนาการ (Exercise & Food Notes)
- เครื่องมือบันทึกตารางการออกกำลังกายรายวันและรายสัปดาห์
- ระบบบันทึกรายการอาหาร ปริมาณแคลอรี และสารอาหารที่ได้รับในแต่ละวัน เพื่อการดูแลสุขภาพที่ถูกต้อง

### 6. ระบบบันทึกการเงินและงบประมาณ (Finance & Budget Tracker)
- บันทึกและติดตามรายรับ-รายจ่ายส่วนบุคคลแบบเรียลไทม์
- ระบบจำลองการวางแผนงบประมาณ (Budgeting) พร้อมประวัติการทำธุรกรรมและกราฟวิเคราะห์ทางการเงิน

### 7. ระบบติดตามพอร์ตการลงทุน (Stocks Portfolio)
- ระบบจำลองพอร์ตการลงทุนส่วนบุคคล
- เชื่อมต่อและดึงข้อมูลราคาสินทรัพย์เพื่อช่วยในการวางแผนกลยุทธ์ทางการเงินและการลงทุน

### 8. ผู้ช่วยปัญญาประดิษฐ์ (AI Assistant)
- ทำงานร่วมกับโมเดลปัญญาประดิษฐ์เพื่อช่วยสรุปบันทึก
- ช่วยปรับปรุงระดับการใช้ภาษา แนะนำแนวทางและขั้นตอนการทำงาน ตลอดจนตอบคำถามแบบเรียลไทม์

### 9. ระบบจัดการและแบ่งปันไฟล์ (File Manager & Tools)
- คลังจัดเก็บข้อมูลและไฟล์ส่วนตัว รองรับการอัปโหลดเอกสารและภาพ
- เครื่องมือแปลงไฟล์และปรับแต่งขนาดรูปภาพเบื้องต้น
- ระบบสร้างลิงก์สำหรับแบ่งปันข้อมูลแก่บุคคลภายนอก (Public Link Sharing) พร้อมตัวเลือกกำหนดเวลาหมดอายุของลิงก์เพื่อความปลอดภัย

### 10. ระบบแจ้งเตือนทาง Telegram และ Cron Job
- บูรณาการระบบแจ้งเตือนผ่าน Telegram API เพื่อส่งข้อมูลแจ้งเตือนที่สำคัญ
- ระบบ Cron Job อัตโนมัติคอยรันประมวลผลเบื้องหลังเพื่ออัปเดตข้อมูลและส่งแจ้งเตือนอย่างแม่นยำ

## เทคโนโลยีที่เลือกใช้ (Technology Stack)

### ฝั่งเซิร์ฟเวอร์ (Backend)
- ภาษา: PHP 8.x (Custom MVC Framework)
- การจัดการสิทธิ์และการยืนยันตัวตน:
  - ระบบยืนยันตัวตนปกติ (Session-based Authentication) พร้อมระบบความปลอดภัยป้องกัน CSRF
  - รองรับการเข้าสู่ระบบด้วย Google Authentication (OAuth 2.0)
- บริการแจ้งเตือน: Telegram API Integration
- ตัวประมวลผลพื้นหลัง: PHP Cron Jobs

### ฐานข้อมูล (Database)
- MySQL / MariaDB (ติดต่อผ่าน PDO Singleton Wrapper เพื่อความปลอดภัยสูงสุดจากการโจมตีประเภท SQL Injection)

### ฝั่งหน้าบ้าน (Frontend)
- โครงสร้างและตรรกะ: HTML5, CSS3, Vanilla JavaScript (ไม่ใช้ Tailwind CSS หรือเฟรมเวิร์กจาวาสคริปต์ขนาดใหญ่ เพื่อความคล่องตัวและความเร็วในการแสดงผล)
- สไตล์การออกแบบ: Neumorphism & Soft UI พร้อมรองรับ 4 ธีมพาสเทลระดับพรีเมียม (Pastel Soft, Pastel Lavender, Pastel Mint/Ocean, Pastel Rose/Peach)

## สถาปัตยกรรมโฟลเดอร์ของโปรเจกต์ (Project Directory Structure)

```text
DayFlow/
├── assets/             # ไฟล์ CSS, JavaScript, รูปภาพ และฟอนต์ที่ใช้ในแอปพลิเคชัน
├── config/             # ไฟล์การกำหนดค่าหลัก เช่น การเชื่อมต่อฐานข้อมูลและการตั้งค่าเซสชัน
├── controllers/        # คลาสตัวควบคุม (Controllers) จัดการส่วนประมวลผลหลักของฟีเจอร์ต่างๆ
├── core/               # โค้ดแกนหลักของระบบ (Router, Request, Response, Auth, CSRF)
├── home/               # หน้าเว็บเพจหลัก
├── install/            # ไฟล์สคริปต์การติดตั้งและเตรียมพร้อมระบบเริ่มต้น
├── models/             # คลาสเชื่อมโยงและประมวลผลฐานข้อมูล (Models)
├── sql/                # ไฟล์สคริปต์ SQL สำหรับสร้างโครงสร้างฐานข้อมูล
├── uploads/            # พื้นที่จัดเก็บไฟล์สื่อและเอกสารที่อัปโหลดโดยผู้ใช้
├── views/              # ไฟล์เทมเพลตมุมมองสำหรับการแสดงผล (Views) แยกเป็นหมวดหมู่ตามฟีเจอร์
├── cron.php            # ไฟล์รันงานเบื้องหลังอัตโนมัติ (Cron Job Script)
├── index.php           # หน้าหลักที่เป็นจุดผ่านเดียวเข้าสู่ระบบ (Front Controller)
└── README.md           # คำอธิบายรายละเอียดโปรเจกต์
```

## วิธีการติดตั้งเพื่อทำงานในเครื่องจำลอง (Local Installation)

### ความต้องการขั้นต่ำของระบบ
- PHP 8.0 ขึ้นไป
- MySQL 5.7 หรือ MariaDB 10.4 ขึ้นไป
- Web Server เช่น Apache (แนะนำ XAMPP หรือ Laragon)

### ขั้นตอนการตั้งค่า

1. **คัดลอกโปรเจกต์ (Clone Project)**
   ทำการดาวน์โหลดซอร์สโค้ดมาไว้ที่โฟลเดอร์ Web Root ของคุณ (เช่น `htdocs` สำหรับ XAMPP)

2. **นำเข้าฐานข้อมูล (Import Database)**
   - สร้างฐานข้อมูลใหม่ใน MySQL (เช่น ชื่อฐานข้อมูล `dayflow`)
   - นำเข้าไฟล์ SQL โครงสร้างระบบจากโฟลเดอร์ `sql/` เข้าไปยังฐานข้อมูลที่สร้างขึ้น

3. **ตั้งค่าไฟล์กำหนดค่าระบบ (Configuration)**
   สร้างไฟล์กำหนดค่าระบบภายในโฟลเดอร์ `config/` ดังนี้:
   
   - **config/config.php**
     ```php
     <?php
     define('BASE_URL', 'http://localhost/DayFlow');
     define('APP_ENV', 'development');
     // ค่าการตั้งค่าอื่นๆ เช่น Google Client ID, App Key ฯลฯ
     ```
   
   - **config/database.php**
     ```php
     <?php
     define('DB_HOST', 'localhost');
     define('DB_USER', 'root');
     define('DB_PASS', '');
     define('DB_NAME', 'dayflow');
     ```

4. **เริ่มการทำงาน**
   เปิดเว็บบราวเซอร์แล้วเข้าใช้งานผ่าน URL ที่กำหนดไว้ เช่น `http://localhost/DayFlow`
