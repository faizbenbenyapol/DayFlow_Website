# Changelog — DayFlow

บันทึกการเปลี่ยนแปลงและการพัฒนาของแพลตฟอร์ม **DayFlow** ระบบบริหารจัดการประสิทธิภาพการทำงาน (Productivity), การวางแผนงาน, บันทึกสุขภาพ และการบริหารจัดการการเงินแบบครบวงจร

---

## [1.0.0] - 2026-05-31

การเปิดตัวเวอร์ชันอย่างเป็นทางการครั้งแรก (**v1.0.0**) ซึ่งประกอบด้วยฟังก์ชันการทำงานที่ครบถ้วน ระบบรักษาความปลอดภัยในการเข้าสู่ระบบ และการปรับปรุงสถาปัตยกรรมการออกแบบส่วนติดต่อผู้ใช้ (UI/UX) ให้มีความทันสมัย สวยงาม และรองรับการแสดงผลบนอุปกรณ์ทุกประเภทได้อย่างสมบูรณ์แบบ (Fully Responsive Design)

### ฟีเจอร์หลัก (Core Features)

*   **Dashboard (ระบบแดชบอร์ดหลัก):** ศูนย์รวมการแสดงผลกิจกรรมประจำวัน, โน้ตย่อเร่งด่วน, สถานะการดำเนินโครงการ รวมถึงวิดเจ็ตแสดงสถิติทางการเงินและสุขภาพ เพื่อให้ผู้ใช้สามารถประเมินภาพรวมรายวันได้อย่างมีประสิทธิภาพ
*   **Notes (ระบบจดบันทึกอัจฉริยะ):** ระบบบันทึกข้อความแบบจัดโครงสร้างบล็อกอิสระ (Text, Link, Checklist) รองรับการปักหมุดบันทึกสำคัญ การจัดหมวดหมู่ผ่านแท็ก และระบบเข้ารหัสข้อมูลบันทึก (Note Encryption) เพื่อรักษาความเป็นส่วนตัวขั้นสูง
*   **Project & Task Management (ระบบบริหารจัดการโครงการ):** บอร์ดบริหารงานรูปแบบคัมบัง (Kanban Board) ที่รองรับการเคลื่อนย้ายการ์ดงานแบบลากวาง (Drag-and-Drop) พร้อมระบบระบุระดับความสำคัญ (Priority) และกำหนดวันส่งมอบงาน (Due Date)
*   **AI Assistant (ระบบผู้ช่วยอัจฉริยะ):** บริการวิเคราะห์ข้อมูลและโต้ตอบด้วยปัญญาประดิษฐ์ (AI) สนับสนุนการสรุปเนื้อหาบันทึก ปรับเปลี่ยนระดับภาษา แนะนำแนวทางการดำเนินงาน และตอบข้อซักถามแบบเรียลไทม์
*   **Finance & Budget Tracker (ระบบบริหารการเงิน):** เครื่องมือบันทึกรายรับ-รายจ่ายส่วนบุคคล ติดตามประวัติการทำธุรกรรมรายวันและรายเดือน พร้อมระบบสรุปวิเคราะห์งบประมาณ
*   **Stocks Portfolio (ระบบติดตามพอร์ตการลงทุน):** ระบบจำลองการลงทุน ติดตามมูลค่าพอร์ตการลงทุนส่วนบุคคล และดึงข้อมูลราคาสินทรัพย์ เพื่อสนับสนุนการวางแผนกลยุทธ์ทางการเงิน
*   **Exercise & Food Notes (ระบบบันทึกสุขภาพ):** เครื่องมือบันทึกตารางการออกกำลังกาย และระบบบันทึกคุณค่าทางโภชนาการรวมถึงแคลอรีประจำวัน เพื่อการส่งเสริมสุขภาพอย่างมีระบบ
*   **File Manager & File Tools (ระบบจัดการไฟล์และเอกสาร):** คลังจัดเก็บข้อมูลส่วนบุคคลสำหรับอัปโหลดเอกสาร พร้อมเครื่องมือจัดการรูปภาพ แปลงไฟล์ และบริการแบ่งปันเอกสารภายนอก
*   **Collaboration & Link Sharing (ระบบการแบ่งปันข้อมูล):** ฟังก์ชันแบ่งปันบันทึกและเอกสารแก่บุคคลภายนอกผ่านเครือข่ายอินเทอร์เน็ต โดยสามารถสร้างลิงก์สาธารณะและกำหนดระยะเวลาหมดอายุของลิงก์เพื่อความปลอดภัย

### การปรับปรุงดีไซน์และธีม (UI/UX & Themes)

*   **Neumorphism & Soft UI Integration:** ปรับเปลี่ยนสไตล์การออกแบบให้มีความหรูหราและมีความเป็นมืออาชีพยิ่งขึ้นด้วยรูปแบบ Soft UI โดยนำชุดเงาละมุน (Soft Shadow และ Recessed Inset Shadow) มาประยุกต์ใช้ เพื่อสร้างมิติและความลึกให้แก่ส่วนติดต่อผู้ใช้อย่างเป็นธรรมชาติ
*   **เพิ่มระบบ 4 ธีมพาสเทลใหม่ (New Pastel Themes):** เพิ่มความหลากหลายในการแสดงผลด้วยชุดสีพาสเทลระดับพรีเมียม ซึ่งผู้ใช้สามารถเลือกใช้งานผ่านหน้าต่างการตั้งค่า (Settings) ประกอบด้วย:
    *   **Pastel Soft (ครีมพาสเทล):** การแสดงผลโทนสีครีมอบอุ่น สบายตา และเรียบง่าย
    *   **Pastel Lavender (ลาเวนเดอร์พาสเทล):** การแสดงผลโทนสีม่วงอ่อน อ่อนโยนและเป็นระบบ
    *   **Pastel Mint/Ocean (เขียวมินต์และฟ้าน้ำทะเล):** การแสดงผลโทนสีเย็น ให้ความรู้สึกสดชื่นและมีพลังสร้างสรรค์
    *   **Pastel Rose/Peach (โรสและชมพูพีช):** การแสดงผลโทนสีอบอุ่นร่วมสมัยและมีเอกลักษณ์เฉพาะตัว
*   **ปรับปรุงหน้าเข้าสู่ระบบ (Authentication UI):** ออกแบบหน้าต่างเข้าสู่ระบบใหม่ทั้งหมดภายใต้แนวคิด Soft UI พร้อมพัฒนาปุ่มสั่งการและแถบประเมินความปลอดภัยของรหัสผ่าน (Password Strength Bar) ให้แสดงผลแบบไดนามิกและสวยงามยิ่งขึ้น

### ความปลอดภัยและการเข้าสู่ระบบ (Security & Auth)

*   **Google Authentication:** พัฒนาระบบยืนยันตัวตนและการเข้าสู่ระบบผ่านบัญชี Google (Google Sign-In) เพื่อเพิ่มความสะดวกรวดเร็วและยกระดับความปลอดภัยด้วยมาตรฐานสากล
*   **ปรับปรุงความปลอดภัยของโครงสร้างรหัส (Core Security & Configuration):** ดำเนินการแยกไฟล์กำหนดค่าสำคัญ ได้แก่ `config.php` และคีย์ความปลอดภัยระบบ `.app_key` ออกจาก Git Repository โดยจัดทำระบบคัดกรองผ่าน `.gitignore` เพื่อป้องกันข้อมูลที่มีความอ่อนไหวรั่วไหลสู่สาธารณะ

---

### ประสิทธิภาพและการใช้งานบนมือถือ (Mobile UX & Responsiveness)

*   **Mobile-First Responsive Adjustments:** ปรับปรุงและแก้ไขการแสดงผลบนอุปกรณ์เคลื่อนที่และแท็บเล็ตให้มีความสมบูรณ์แบบยิ่งขึ้น ครอบคลุมแถบเมนูหลัก แดชบอร์ด และหน้ารายละเอียดโน้ต
*   **Notes Block Menu & Interactions:**
    *   ปรับปรุงปุ่มเพิ่มบล็อกบนอุปกรณ์เคลื่อนที่ให้แสดงผลเป็นเมนูป๊อปอัป (**Block Type Menu**) พร้อมสัญลักษณ์ไอคอน SVG แยกตามประเภทบล็อก (Text, Link, Checklist) อย่างชัดเจน
    *   ปรับปรุงค่ามิติ ความกว้าง ความสูง และคุณสมบัติ CSS Touch-friendly เพื่อรองรับการควบคุมผ่านระบบสัมผัสและการเลื่อนหน้าจอที่เสถียรบนอุปกรณ์พกพา
    *   ปรับแต่งการแสดงผลของบล็อกลิงก์ (Link Block) และบล็อกรายการตรวจสอบ (Checklist Block) ให้มีความเป็นระเบียบและใช้งานง่ายบนหน้าจอขนาดเล็ก
*   **ยกระดับความเร็วในการประมวลผล:** ปรับปรุงประสิทธิภาพของไฟล์ CSS และ JavaScript ส่วนกลางเพื่อให้ระบบโหลดข้อมูลได้รวดเร็วขึ้น พร้อมทั้งพัฒนาการตอบสนองเชิงฟิสิกส์ของการกดปุ่มและไอคอน (Feedback Transitions) ให้มีความลื่นไหลเป็นธรรมชาติ

### การแก้ไขข้อผิดพลาดและการปรับปรุงเสถียรภาพ (Bug Fixes & Refactoring)

*   **Response Core:** ปรับแต่งโครงสร้างระบบภายในของ `Response.php` เพื่อรับรองการส่งผ่านข้อมูล Header และการกำหนดโครงสร้าง Content-Type ให้ถูกต้องตามมาตรฐานเว็บ
*   **Profile Settings Saving:** แก้ไขปัญหาทางเทคนิคเกี่ยวกับการบันทึกตัวเลือกธีมและข้อมูลประวัติส่วนบุคคลในระบบ `SettingsController.php`
*   **Share Page View:** แก้ไขการแสดงผลหน้ารายงานลิงก์หมดอายุ (`views/share/expired.php`) และหน้าตรวจสอบการเข้าถึงข้อมูลร่วมกัน (`views/share/index.php`) ให้มีโครงสร้างการแสดงผลแบบ Responsive ที่เป็นระเบียบ สวยงาม และสอดคล้องกับธีมหลักของระบบ
